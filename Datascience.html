Introduction to data science

Embark on a journey of discovery with our Introduction to Data Science course, designed specifically for high school students. Dive into the fascinating world of data analysis, where you'll learn to extract insights from vast datasets using powerful tools like Google Sheets, Python, and Data Commons. This course is not just about learning; it's about becoming a data explorer, developing your skills in data analysis, sampling, correlation/causation, and much more. As you progress, you'll gain a deeper understanding of the role of data in society and how it can drive decision-making and innovation. By the end of this course, you'll have a portfolio showcasing your data science work, demonstrating your newly acquired abilities. Join us and unlock the power of data science!

What You will learn

  • Data Exploration: Learn how to explore and understand data using Python and libraries like Pandas and NumPy.
  • Data Preprocessing: Gain skills in cleaning and transforming data to prepare it for analysis and modeling.
  • Data Visualization: Master the art of visualizing data using libraries like Matplotlib and Seaborn to uncover insights.
  • Descriptive Statistics: Understand the basics of descriptive statistics to summarize and describe data sets.
  • Machine Learning Algorithms: Dive into regression, classification, clustering, and dimensionality reduction algorithms.
  • Model Evaluation: Learn how to evaluate the performance of machine learning models using metrics like accuracy, precision, recall, and F1 score.
